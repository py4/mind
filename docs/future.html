About This Page

I have a fetish for minimalism. I've owned multiple Wordpress blogs before. The last one was a truly minimal one to write my own monologs. Later on I turned to static-site generators because I felt a server/database is an overkill for hosting a simple blog. I tried both Jekyll and Hugo for my blogs. I'm currently happy with hugo. However, recently I felt an urge toward more extremism in the wonderland of minimalism: Minimizing visual beauty and CSS modifications. Consequently I created this page to practive this new fetish and also express myself in English to target a broader class of audience.

- It's Pooya and I write raw HTML here.
